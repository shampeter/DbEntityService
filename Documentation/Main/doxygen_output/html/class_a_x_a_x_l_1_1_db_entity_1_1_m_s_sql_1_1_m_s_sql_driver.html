Select multiple entities based on parameters where the key is a property name and value is a list of object. This is used in retrieving the same type of children from multiple parents at the same time which is faster than retrieving the children on multiple parents one by one.

TEntity objec type
Parameters

nodeThe Node representing the meta data and object-to-relational database mapping of the entity class.
parametersDictionary of name to list of values representing the where condition, assuming AND operation on all key-value pairs.
whereClausesAdditonal where clause in addition to the parameters
orClausesGroupAdditional or clauses in addition to the parameters

childInnerJoinOrClausesGroupInner joins to childset and children selection or group condition
