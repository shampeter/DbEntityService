IDbService AXAXL.DbEntity.Interfaces.IDbService.Bootstrap ( Assembly[]  assemblies = null,
string[]  assemblyNamePrefixes = null 
)

Boostrap DbService where the servie will scan assemblies for class type which has the a System.ComponentModel.DataAnnotations.Schema.TableAttribute defined. and create AXAXL.DbEntity.EntityGraph.Node to store the discovered meta data. if assemblies is null or empty, then use all loaded assemblies as found in AppDomain.CurrentDomain. Use assemblyNamePrefixes to narrow down the assembly to start by specifying the assembly name prefix. Service will use case insensitives match to test assembly by name with string.StartsWith(string, StringComparison) If assemblyNamePrefixes is null or empty, service will search through all assemblies specified.

Parameters
assemblieslist of assemblies to scan. If none, full assemblies loaded will be used. AppDomain.CurrentDomain
