Build up meta data on entity objects found in assemblies or all loaded assemblies as found in AppDomain.CurrentDomain if assemblies is null or empty. Use assemblyNamePrefixes to narrow down the assembly to start by specifying the assembly name prefix. Service will use case insensitives match to test assembly by name with string.StartsWith(string, StringComparison) If assemblyNamePrefixes is null or empty, service will search through all assemblies specified.

Parameters
assembliesOptional. List of assemblies to search for meta data on entity class.
assemblyNamePrefixesName prefixes.
